## 《東方逍遥鎭~TouHou Getaway》  
你说的对，但是《东方逍遥镇~TouHou Getaway》是由白玉楼自主研发的一款全新弹幕射击游戏。  
游戏发生在一个被称作「逍遥镇」的幻想世界，   
在这里，被神主选中的人将被授予「自机」，导引符卡之力。    
你将扮演一位名为「博丽巫女」的神秘角色，在自由的旅行中邂逅性格各异、能力独特的同伴们，  
和他们一起击败强敌，拯救陷入轮回的村镇——同时，逐步发掘「灾变」的真相。
